

# Beoble

You can check Demo [here](https://beoble-toolkit.vercel.app/)

### Prerequisite

You should have Nx installed globally
```js

// using npm
npm install -g nx

// using yarn
yarn global add nx
```

run project after cloning

```javascript
yarn

// start storybook
nx run react:storybook

// start example
nx serve example
```

build project
```js
// build js-sdk
nx build js-sdk

// build react
nx build react
```
